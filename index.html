Guide de prise en main

1. ACCUEIL (Recherche rapide)

  • Sélectionnez un BRAS → Boutons bleus apparaissent
  • Ville (optionnel) → Affinez votre recherche
  • Recherchez par mot directeur (ex: "paix", "republique")
  • Résultats groupés par ville + Type 2 si pas de Type 1

2. RECHERCHE VOCALE

  • 🎤 Micro → Dites “RUE DE LA PAIX” et seul le mot directeur sera conservé : “PAIX”.
  • Popup confirme → Confirmer lance recherche
  • Réessayer si erreur, Annuler pour arrêter

3. CAMERA (OCR)

  • Activer toggle "Reconnaissance visuelle" (⚙️)
  • 📷 Caméra → Viser numéro + Nom rue
  • Région verte → Capturer → Recherche automatique

4. PARAMÈTRES (⚙️)

  • 📥 Importer Excel → Fichier adresses (.xlsx/.xls)
  • 📤 Exporter Excel → Sauvegarde complète des adresse
  • + Ajouter adresse → CRUD complet (ajout/modif/suppr)
  • 📓 Journal → Trace toutes modifications (export Excel)
  • 🗑️ Effacer → Reset données locales

5. IMPRESSION (PDF/Word)

  • Dans chaque BRAS → Bouton 🖨️ Imprimer
  • Sélectionnez villes → Couleurs pastilles personnalisables pour les équipes
  • PDF (A3 / A4) ou Word (.docx)
  • Numéros spéciaux (CS, PICKUP, PPDC, REEX, CI) en rouge/surlignés

6. TYPES DE RECHERCHE

  • Type 1 (T1) → Adresses exactes/réelles
  • Type 2 (T2) → Alternatives/approximatives (fallback)

7. PERSONNALISATION

  • Thème sombre → Toggle en haut ⚙️
  • Couleurs villes → Pastilles dans popup Impression

ASTUCES PRO

  • Recherche floue → 2-3 lettres suffisent
  • Numéros spéciaux → ROUGE (CS, PICKUP, PPDC, REEX, CI)
  • Sauvegarde auto → LocalStorage persiste entre sessions
  • Responsive → Tactile + clavier OK
  • Permissions → Micro/Caméra HTTPS requis

NOTES

  • Fichiers Excel : "Colonnes BRAS, Ville, Adresse, Numéro de tournée, Type Recherche"
  • Recherche insensible accents/majuscules
  • HTTPS obligatoire pour micro/caméra
